# 안면 특징점 추적 및 객체 합성 프로젝트

## 프로젝트 소개
- 목적: MediaPipe를 활용하여 실시간으로 안면 특징점을 추출하고, OPENCV를 통해 가상 객체(사슴뿔, 코)를 합성
- 주요 기술: Python, OpenCV, MediaPipe

## 실행 결과 
[결과 GIF](output.gif) ## 주요 구현 로직
1. 특징점 추출: MediaPipe Face Detection을 사용하여 눈, 코 등 6개의 주요 랜드마크의 상대 좌표 획득.
2. 좌표 매핑: 상대 좌표를 영상 해상도에 맞는 픽셀 좌표로 변환 및 보정.
3. 알파 블렌딩: 4채널(BGRA) 이미지를 활용하여 배경과 합성하는 'overlay' 함수 설계

## 보완해야 할 점
1. 2D 이미지 매핑 방식으로 인한 고개 회전 시 왜곡 및 부자연스러움.
2. 실시간 카메라 영상을 활용하여 프로젝트를 발전.
